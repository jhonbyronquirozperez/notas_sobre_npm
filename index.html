Comandos Importantes

  • npm init - Inicializa un nuevo proyecto con un archivo package.json.
  • npm install paquete - Instala un paquete específico.
  • npm uninstall paquete - Desinstala un paquete.
  • npm run script - Ejecuta un script definido en package.json.

Ejemplos de Código


// Ejemplo de un archivo package.json básico
{
  "name": "mi-proyecto",
  "version": "1.0.0",
  "scripts": {
    "start": "node app.js"
  },
  "dependencies": {
    "express": "^4.17.1"
  }
}
